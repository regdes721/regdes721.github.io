About

I am a programer that wants to create beautiful things in the world.

After spending most of my career working in Customer Service and Tech Sales, I realized that I would be much happier in my professional life if I combined my desire to work in the tech industry with my creative impulses (namely my interests in art and writing). I find software engineering an effective means to experience this happiness that I seek, especially because I am able to witness firsthand the transformation of an application from mere ideas captured in user stories and wireframes to a fully fledged website that people can enjoy experiencing. I also appreciate how much I am able to learn from fellow software engineers on how to better structure my own code and improve the user experience.

As a full stack software engineer, I have enjoyed using the following technologies and am willing to learn more!

PixelChat

PixelChat is a website clone of Slack populated with content inspired by famous video game characters like Mario, Minecraft Steve, Kirby, and Zelda. PixelChat allows users to create and join unique servers and channels designed to host fun conversations between server members. To help express themselves, users can customize their profile and add emoji reactions to any message they like!
